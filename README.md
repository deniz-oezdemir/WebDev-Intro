# WebDev-Intro

## Summary
These are implementations following the introductory course on web development taught by Brian Holt on Frontend Masters. The course covers various topics related to web development, including HTML, CSS, JavaScript. It provides a comprehensive overview of the fundamentals and best practices in web development.
